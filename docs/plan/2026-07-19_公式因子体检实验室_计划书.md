## 2. 方法论 S0-S5(方法论主体,文档化)

### S0 前置检查(不达标拒跑并说明)
- baseline `data/baseline/{formula}_selections_{tag}.parquet` 存在;
- 信号总量 ≥200 条、且有效截面日(当日信号 ≥5 条)≥40 天 —— 小信号公式 IC 无统计意义,明确拒跑;
- 双窗口参数齐全(`--tag` 短窗 + `--tag2` 长窗),缺 tag2 警告"单窗口结论不作数"。

### S1 数据就位
kline 面板(v3 缓存)、daily_basic、融资余额、板块映射、指数序列;事件因子可选(`--skip-event-factors` 透传)。

### S2 IC 筛选
41 因子 × 3 期限(5/10/20)× **强制双窗口**,产出 `factor_ic_{formula}_{tag}.csv`。

### S3 族归纳 + 自动选臂(★核心新增,选臂规则预注册)
1. **族归属内置于注册表**(v2 修订,单一数据源):`factor_ic_screen.py` 的注册表每行带 `family` 字段(加因子仍只加一行,族随行),不维护第二份映射;族别:过热反转 / 规模 / 价值 / 板块 / 事件 / 情绪 / 指数关联 / 日内结构 / 其他;
2. 族内取 |ICIR| 最大者为**族代表**;
3. 入选条件(v2.1 校准):阈值只在**证据窗**(长窗)卡(|IC|≥0.03 且 |ICIR|≥0.3),其余窗口只要求 **IC 符号一致**;双窗口一致性由 S4 A/B 终审强制执行,不在 S3 重复卡阈值(QUANTQQ 实战发现:1y 窗效应系统性偏弱,双窗同阈值会把已知能过 A/B 的族错杀);
4. 取满足条件的前 2 族,各配 2 档剔除(top10% / top20%),**剔除方向由 IC 符号自动定**:IC<0 → 剔高端(如过热);IC>0 → 剔低端(如小市值);
5. **零族达标路径**(v2 新增):无任何族满足条件 → 不跑 S4,报告输出"无可行动因子,建议保持现状"——这是合法结论,不硬凑 A/B;
6. 选臂结果打印并写入报告(S3 全程可审计,防挑数据)。

### S2 产出( v2 新增贯通件)
除 CSV 排行榜外,S2 额外落盘**信号×因子矩阵** `output/gs_filter/factor_matrix_{formula}_{tag}.parquet`(每行 = 一条信号,列 = 全部因子值 + fwd 收益)。S4 直接消费该矩阵,不再自行算因子——因子值单一来源,杜绝两处各算一遍的 drift。

### S4 A/B 终审
每窗口:基线 + 2 族 × 2 档 = 5 臂回测(同 stop_config),判定沿用预注册口径:
Calmar 不差于基线 0.1 且年化提升 → PASS(增收);Calmar 不差且回撤改善≥2pp、年化损失≤1pp → PASS(控回撤)。
**双窗口同臂均 PASS 才算最终通过**;单窗口 PASS 标"待复核"。

### S5 报告
固定模板(markdown):总榜摘要 → 族格局表 → S3 选臂记录 → S4 终审表 → 判定结论 → 可信度警告(回测系统性乐观 + 池子条件化 + 多重检验边界)→ 附录(口径/覆盖率)。
落盘 `docs/audit/{日期}_{formula}_{tag}_{tag2}_因子体检报告.md`(带窗口标签,防重跑覆盖)。

## 3. 五条纪律(写入方法论文档,违反即报告作废)

1. **IC 是海选不是终审** — 任何过滤规则进策略前必须过 S4;
2. **双窗口一致才算数** — 单窗口结论一律标"待复核";
3. **数族不数因子** — 防多重检验挖矿;
4. **因果测试必须过** — 每个新因子自带"截断未来,过去不变"测试;
5. **报告必带可信度警告** — 回测系统性乐观(CLAUDE.md 铁律) + 结论仅对该公式信号池成立 + 批量跑公式存在又一层多重检验。

## 6. 边界与风险(诚实清单)

| 项 | 说明 |
|---|---|
| 结论池子条件化 | IC/终审都在该公式信号池内成立,报告模板强制写明"仅对 {formula} 信号池" |
| 批量跑的多重检验 | 跑 N 个公式挑最好的"公式×过滤"组合 = 又一层数据挖掘;纪律:跨公式结论需样外验证或仅作研究参考 |
| 算力预算 | S4 每公式 2 窗口 × 5 臂 = 10 遍回测 ≈ 30-50 分钟;S2 缓存复用后不重复拉数 |
| 数据覆盖边界 | 北向日频已断供(2024-08 起)、两融/陆股通覆盖子集、事件因子仅 1 年窗 —— 报告如实标覆盖率,不硬撑 |
| 小信号公式 | S0 直接拒跑,不产出伪结论 |
| 引擎口径 | 终审继承回测系统性乐观(7.5/10),相对排序可参考,绝对年化不可全信 |

## 7. 明确不做(本期范围外)

- L3 web 产品化(页面/按钮);
- 质量/成长因子(ROE 等,需公告日对齐防前视)——二期;
- 新因子族扩充(注册表加一行的事,想用随时加,不堵死);
- 跨公式横向对比排名(多重检验风险,一期只做单公式体检)。
