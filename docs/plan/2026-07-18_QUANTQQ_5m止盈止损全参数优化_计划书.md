> 日期: 2026-07-18 | 触发: 用户指令"移动止盈优先, 全参数组合, 2024-08-01 至今, 5m 精度, 年化>23%, 启用 LOOP, 允许多 AGENT"

## 1. 目标与验收标准

| 项 | 值 |
|---|---|
| 公式 | QUANTQQ (选股 1d, 前复权) |

## 2. 用户已拍板的决策点

1. 股票池 = 全 A 股 (不用上证50/300, 不沿用 strategy yaml)
2. 资金口径 = 300万/单票 2万 (v4 报告最可信档, Calmar 9.52)

4. 搜索策略 = 粗扫→细化两阶段 (不做全量笛卡尔积穷举)

## 3. 技术方案

### 3.1 驱动架构 (tools/quantqq_5m_sweep.py, 新独立文件)

- **prep (一次性)**: StockSelector 选股 → `DataFetcher.get_kline_windowed` 5m 稀疏窗口取数 (窗口 60 交易日 > 最大持仓 40 天+15 缓冲, 引擎铁律) → 复刻 `engine.run()` line 776-855 矩阵准备 → 涨停过滤预做 (stop 无关) → float64 npy 落盘
- **run (每组合)**: mmap 加载矩阵 → `engine.run_cached()` (filter_limit_up=False) → 指标追加 CSV → 断点续跑
- **并行**: subprocess 分片 (`--shard i --nshards N`), 遵守 CLAUDE.md 批量约定; mmap 让多进程共享 OS 页缓存
- 选股/矩阵缓存幂等, 重跑不重取数

### 3.2 为什么用 run_cached 不用 run

- run_cached 预取数据复用, 单组合 ≈ 引擎循环耗时 (审计实测 120万 bar ≈ 76ms 量级外推)
- 代价: **run_cached 不支持 degrade_5m** (缺 5m 的股-天丢信号, 计划书 LOW-3 已接受; no_data 名单仅 8 只)
- 一致性: 矩阵准备逐行复刻 engine.run(), 同 helper 同顺序, 防口径分叉

### 3.3 粗扫网格 (2592 组合, priority 固定 trailing_first)

| 维度 | 档位 | 说明 |
|---|---|---|
| 硬止损 cost | -8% / -12% / -20% | 覆盖紧/中/宽 |
| 移动止盈激活 act | 2% / 3.5% / 5% / 8% | v4 显示 1.5-3.5% 区最肥 |
| 移动止盈回撤 dd | 0.5% / 1% / 2% | "上涨多少后回撤多少" |
| 阶梯止盈 | 8 种结构 | 关 / 单档 5%-8%-12% 全卖 / 双档 5+12·30%、6+15·30% / 三档 4-10-20·25% / 四档 4-8-15-25·20% (v4 冠军结构) |
| 时间止损 | 12 / 20 / 40 天 | 无条件持仓天数 |
| 条件时间止盈 | 关 / (7天,1.5%) / (15天,3%) | 有条件天数 |

### 3.4 细化阶段

粗扫 Top 区域逐维加密 (如 act 1.5%-4% 步长 0.5%, dd 0.3%-1.5% 步长 0.25%, cost -4%~-10%), 并补**卖出比例维度** (粗扫只固定 8 种结构的 ratio, 细化对热区结构扫 sell_ratio 30%/50%/70%)。组合数视热区大小而定, 通常 1000-3000。

### 3.5 排除维度及理由 (审计 M4)

| 维度 | 状态 | 理由 |
|---|---|---|
| first_day 首日未达标 | 排除 (enabled=False) | 与"T 日 15:00 收盘买入"口径重叠, 首日几乎无持仓时间可评估 |
| ATR 止损 (atr_stop) | 粗扫排除, 细化候选 | 网上调研显示 ATR/Chandelier 法优于固定百分比; 但它是独立止损体系, 与固定%网格不可直接比, 放细化阶段作"新方法变体"单独测 |
| formula_sell 公式卖出 | 排除 | 依赖 TDX 公式矩阵预计算, 与 5m 窗口路径未打通; 且卖出公式未确认真名 |
| priority 其他两档 | 排除 | 用户明确"移动止盈优先", 固定 trailing_first |
| 保本止损 breakeven | 明确不加 | 网上调研实证一致负 EV (Liverium/Mabe 多组回测) |

### 3.6 LOOP 终止条件

满足以下全部即停:
1. ≥1 组合 年化>23% 且 回撤≤15%
2. 该组合邻域 (每维 ±1 档) 中 ≥50% 也年化>20% — 证明不是孤点
3. 交易笔数 ≥ 1000 (统计显著)

若不满足: 分析短板维度 → 调整网格 → 再跑 (上限 5 轮防死循环)。
