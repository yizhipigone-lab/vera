## 1. P0 — 让尺子重新可信（5/5 完成）

### P0-1：删除 bfill 前视 + fill_data=False

**问题**：`engine.py` 用 `.bfill()` 把未来价格回填到停牌日，是教科书级前视偏差。且 `get_kline` 默认 `fill_data=True`，TDX 源头已填 NaN，删 bfill 变成空操作。

**修复**：
| 位置 | 改动 |
|---|---|
| `engine.py:368` | `close[cols].ffill().bfill()` → `close[cols].ffill()` |
| `engine.py:376` | `high_df.reindex(...).ffill().bfill()` → `.ffill()` |
| `engine.py:385` | `low_df.reindex(...).ffill().bfill()` → `.ffill()` |
| `engine.py:356` | `get_kline(...)` → `get_kline(..., fill_data=False)` |

**验收**：停牌日 NaN 显式暴露，ffill 仅前向填充做 mark-to-market，不再出现未来价成交。
