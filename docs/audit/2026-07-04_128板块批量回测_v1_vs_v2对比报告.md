## 7. 风险与自检提醒（规则 2）

### 7.1 高收益但伴随回撤扩张

半导体板块 +27.59% 看起来亮眼，但**最大回撤从 -0.45% 涨到 -6.14%**（13 倍）。这种"高信号密度 + 高收益 + 中等回撤"在样本外极易回撤。**强烈建议**：样本外（2026-07-04 之后）实盘验证前不要裸跑。

### 7.2 ⚠️ 【2026-07-04 23:25 修正】trades=0 不是引擎 bug，是脚本读错键名

**原报告说**：所有板块 trades=0 → 怀疑 high-T3（端到端未跑）问题坐实

**实测后修正**：
- 真实根因：`_run_sector_batch_v2.py:79` 写 `bt.get("trade_count", 0)`
- 但 `BacktestEngine.run()` 返回的 trade 数在 **`metrics["total_trades"]`** 里，不在顶层 `trade_count`
- 所以脚本读到 `None → 0`，所有板块报表的 trades 列被写成 0
- **引擎本身完全正常**：601872 实测跑出 2 笔真实交易（已有 4 个端到端测试守护，见 §11）

**重跑 v3 数字（trades 字段修正后）**：

| 项 | v2（tradesbug）| v3（trades fixed）| Δ |
|---|---|---|---|
| 累计交易数 | 显示 0 | **13,126** | 真实数字 |
| 半导体板块交易数 | 显示 0 | **568** | 真实 |
| 通用设备板块交易数 | 显示 0 | **578** | 真实 |
| 半导体信号数 | 355 | 355 | 一致 |
| 半导体信号→交易转化率 | n/a | **160%** | 信号多次进出 |
| **所有 cumret 数值** | **真实** | **真实** | **Δ = +0.00pp（逐板块完全一致）** |

**重要结论**：本报告 §3 / §5 / §6 中所有 cumret / annret / maxdd / sharpe / winrate 数字**全部真实**，与 v3 一致。**只有 trades 列显示 0**，已修复。

### 7.3 公式稳定性

化学制品从 -0.14% → +10.11%，中药从 -0.97% → -5.38%，公式调整**改变了行业相对排序**。建议把 v2 公式的应用场景锁定在 Top 10 板块集中，避免在 Bottom 5 板块中使用。
