# 公式因子体检方法论

> 版本: v1(2026-07-19) | 驱动工具: `tools/formula_lab.py` | 计划书: `docs/plan/2026-07-19_公式因子体检实验室_计划书.md`
> 用途: 任意选股公式 → 一条命令 → 得出"该公式该配什么过滤规则"的判定结论。

## 一句话

**以公式信号池为底座,批量测因子与未来收益的相关性(IC 海选),把最强因子族做成过滤规则,再用同历史双跑回测(A/B 终审)判定是否采纳。** IC 是海选,A/B 是终审,两级缺一不可。

## 流程 S0-S5

| 阶段 | 内容 | 关键规则 |
|---|---|---|
| **S0 前置检查** | baseline selections 存在性 + 信号量检查 | 信号 <200 条或有效截面日(≥5条/日) <40 天 → **拒跑**,不产出伪结论 |
| **S1 数据就位** | kline 面板 / daily_basic / 融资 / 板块映射 / 指数 | 全部缓存复用;事件因子缺失窗口自动降级 |
| **S2 IC 筛选** | 41 因子 × 3 期限(5/10/20)× 双窗口 | 日频截面 Rank IC + ICIR + 五分位价差;产出排行榜 CSV + 信号×因子矩阵 |
| **S3 族归纳选臂** | 族内取证据窗 \|ICIR\| 最大者为族代表 | 预注册:阈值只卡**证据窗**(长窗 fwd10 \|IC\|≥0.03 且 \|ICIR\|≥0.3),其余窗口只要求 IC 符号一致(双窗口一致性由 S4 终审强制执行);取前 2 族 × 2 档;**零族达标 → 直接出"保持现状"结论,不硬凑 A/B** |
| **S4 A/B 终审** | 基线 + 自动臂,同 stop_config 回测 | 剔除方向由 IC 符号定(负 IC 剔高端,正 IC 剔低端);判定:Calmar 不差于基线 0.1 且年化提升 → PASS(增收);Calmar 不差且回撤改善≥2pp、年化损失≤1pp → PASS(控回撤) |
| **S5 报告** | 固定模板,落 `docs/audit/` | 含 S3 选臂记录(可审计)+ 终审表 + 判定 + 可信度警告 |

## 五条纪律(违反即报告作废)

1. **IC 是海选不是终审** — 任何过滤规则进策略前必须过 S4;
2. **双窗口一致才算数** — 单窗口结论一律标"待复核";
3. **数族不数因子** — 同一信息源的 10 个马甲因子算 1 个证据,防多重检验挖矿;
4. **因果测试必须过** — 每个新因子自带"截断未来数据,过去取值逐值不变"测试(tests/test_factor_ic.py);
5. **报告必带可信度警告** — 回测系统性乐观 + 结论仅对该公式信号池成立 + 批量跑公式存在又一层多重检验。

## 边界(诚实清单)

- **结论池子条件化**:IC/终审都在该公式信号池内成立。"过热反转"是 QUANTQQ(动量公式)的体检结果,换个反转公式图景可能完全不同——这不是缺陷,是体检的意义。
- **批量使用的过拟合循环**:跑 N 个公式挑最好的"公式×过滤"组合 = 又一层数据挖掘;跨公式结论需样外验证或仅作研究参考。
- **算力预算**:S4 每公式 2 窗口 × 5 臂 = 10 遍回测 ≈ 30-50 分钟;S2 缓存复用后不重复拉数。
- **数据边界**:北向日频 2024-08 起断供;两融/陆股通仅覆盖子集;事件因子仅近 1 年窗。
- **引擎口径**:终审继承回测系统性乐观(7.5/10),相对排序可参考,绝对年化不可全信。终审默认 1d 口径(2026-07-20 用户拍板接受);移动止盈类紧参数(回撤 ≤0.3%)在 1d bar 上偏乐观,终审结论按相对排序读。

## 如何加新因子

`tools/factor_ic_screen.py` 注册表加一行(名称, 函数, 是否需要ctx, **族**),并在 `tests/test_factor_ic.py` 的因果性参数列表加一行。族归属随行,S3 自动纳入对应族,无需改其他任何代码。

## 首期实战样本(2026-07-19, QUANTQQ)

- 41 因子双窗口筛选:"过热反转"为唯一强族(14 因子互证, IC -0.10~-0.18);
- S4 终审:`turnover_rate top10` / `dist_ma20 top10` / `dist_ma20 top20` 三臂**双窗口 PASS(增收)**(1y Calmar 4.02→6.94;3y 年化 -37.6%→+2.4%);
- 附带发现:`strategy_QUANTQQ.yaml` 止损参数仍是旧"先生前端参数",非 5m 优化冠军结构(配置漂移,见当日 A/B 报告)。
